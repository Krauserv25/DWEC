RACISTA 1
Para que las fichas no estén tristes deben ser colocadas junto a otras tres de su mismo color como mínimo. Al situar el puntero del ratón sobre una de las fichas tristes se representarán las ubicaciones dónde ésta pueda colocarse para ser feliz. Para que las fichas no estén tristes deben ser colocadas junto a mínimo, una ficha del color opuesto. Al situar el puntero del ratón sobre una de las fichas tristes se representarán las ubicaciones dónde ésta pueda colocarse para ser feliz. Este juego consiste en lograr más fichas del color correspondiente que el rival.
Para ello es necesario cerrar una fila del color contrario con dos del color del jugador para convertir las fichas interiores a ese color. Solo será posible colocar fichas ahí donde se pueda cerrar o convertir colores a favor.
